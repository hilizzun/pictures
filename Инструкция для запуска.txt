Инструкция для Windows:
1) Откройте терминал и перейдите в корневую директорию проекта
2) Введите команду pip install auto-py-to-exe для установки необходимой библиотеки
3) Введите команду pip install -U pyinstaller для установки необходимой библиотеки
4) Введите команду pip install -r requirements.txt
5) Введите команду pyinstaller main.py
6) Перейдите в папку с exe файлом, введя команду cd .\dist\main
7) Введите команду start main.exe для запуска программы

Инструкция для Linux:
1) Установите Anaconda с официального сайта
2) Запустите файл install.sh для создания виртуальной среды, установки необходимых библиотек
3) Запустите файл start.sh для активации виртуальной среды и запуска программы
