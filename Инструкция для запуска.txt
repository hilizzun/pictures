1) Откройте терминал и перейдите в корневую директорию проекта
2) Введите команду pip install pyinstaller для установки необходимой библиотеки
3) Введите команду: pyinstaller --onefile main.py
4) Перейдите в папку с exe файлом, введя команду: cd .\dist
5) Введите команду start main.exe для запуска программы
