1) Откройте терминал и перейдите в корневую директорию проекта
2) Введите команду pip install -r requirements.txt
3) Введите команду pip install pyinstaller для установки необходимой библиотеки
4) Введите команду: pyinstaller --onefile main.py
5) Перейдите в папку с exe файлом, введя команду: cd .\dist
6) Введите команду start main.exe для запуска программы
